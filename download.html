Download incomplete / app won’t start
  1. Extract the full .zip — run RawGraded Studio.exe or RawInvestor.exe at the top level of the extracted folder (not inside a nested subfolder).
  2. Delete older RawGraded-Studio-* / RawInvestor-* copies in Downloads before re-downloading.
  3. Right-click the .zip → Properties → size must match the bytes shown above (Studio: __STUDIO_UNPACKED_BYTES__, Investor: __INVESTOR_UNPACKED_BYTES__).
  4. Upload/download in binary mode (FTP). ASCII mode corrupts archives.
  5. SmartScreen → More info → Run anyway. Add rawgraded.com to antivirus exceptions if needed.
  6. Portable: you can move the extracted folder or run from USB; do not delete files from the folder while the app is open.

RawGraded Studio — how setup works

Extract the portable .zip and run RawGraded Studio.exe. Studio guides you through AI setup on first launch.

  1. Download & run (portable)

    Download RawGraded-Studio-1.0.0-win-unpacked.zip, extract to any folder, and run RawGraded Studio.exe at the root of the extracted files. No installer required.

  2. NSIS installer (optional)

    Prefer a wizard? Use RawGraded-Studio-1.0.0-Setup.exe instead. On the Local AI (optional) page you can enable Ollama install and Download vision model on first launch (~4–8 GB, one-time download).

  3. First launch — bootstrap

    If you queued a model download, Studio opens a bootstrap screen: it may install or start Ollama and run ollama pull llama3.2-vision. Wait for the log to finish, then click Continue to Studio.

  4. AI Launcher — pick a provider

    If bootstrap was skipped or you need to reconfigure, the AI Launcher appears:

    • Local & private (Ollama) — recommended; free on your PC
    • Cloud (Google Gemini) — requires an API key from Google AI Studio

  5. Ollama checklist

    For local AI, confirm each line turns green: Ollama installed → Ollama running (port 11434) → Vision model: llama3.2-vision. Use Install Ollama, Start Ollama, or Download model as needed, then Continue to Studio.

  6. Studio workflow

    Upload or capture front/back → crop → PSA-style centering → AI grading → view and export your certificate. Settings and grading history are stored locally on your PC.

Bootstrap or Ollama failing? Click Use Gemini (cloud) instead, enter your API key, and continue — you can switch back to Ollama later in Studio settings.
