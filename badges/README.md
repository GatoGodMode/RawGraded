# Badge Icons

Upload badge icons here. These will be accessible at:
`/assets/badges/[filename]` on your hosted RawGraded site (e.g. `https://rawgraded.com/assets/badges/[filename]`).

## Usage
When creating a badge in the Admin Dashboard, use:
- Icon URL: `/assets/badges/first_scan.png`
- or full URL: `https://rawgraded.com/assets/badges/first_scan.png`

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG
- SVG
- GIF
- WEBP
