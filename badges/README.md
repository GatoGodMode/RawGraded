# Badge Icons

Upload badge icons here. These will be accessible at:
`https://rawgraded.com/badges/[filename]`

## Usage
When creating a badge in the Admin Dashboard, use:
- Icon URL: `/badges/first_scan.png`
- or full URL: `https://rawgraded.com/badges/first_scan.png`

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG
- SVG
- GIF
- WEBP
