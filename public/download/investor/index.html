After you install (extension-first)

  1. Load the MV3 extension — Edge: enable Developer mode, then Load unpacked and select resources\browser-extension-edge next to RawInvestor.exe (installer, portable, or unpacked folder). Open INSTALL-MICROSOFT-EDGE.txt inside that folder for Edge steps; Chrome: same folder via chrome://extensions → Load unpacked.
  2. Point the extension at the API — In the extension popup, set Server base to http://127.0.0.1:3001 (or the port your build uses). If the server uses hardened tokens, enter the same Extension token / App token in the popup.
  3. Optional sold comps — In the app: Admin → Configuration → eBay API for application OAuth (Marketplace Insights). This complements listing capture; it does not replace a normal browser session for live pages.
  4. Arsenal — Open Arsenal from the workspace row; it reads the same merged SQLite portfolio as extension captures (POST /api/ext/capture).

More detail: in-app User Guide (Chrome / Edge extension section) and the project README for token and path notes.
